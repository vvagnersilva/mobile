Dados da GRU
O Número de Referência é o campo identificador do pagamento. Este campo deverá ser suficiente para fazer a correspondência entre a solicitação e o pagamento. Para serviços de "Diária e/ou remoção de veículos" este campo deverá ser preenchido com o número do Recibo de Recolhimento do Veículo (e-DRV). Para serviços de "Diária e/ou remoção de animais" este campo deverá ser preenchido com o número do Recibo de Recolhimento de Animal (RRA). Para serviço de "Credenciamento de empresa de escolta" este campo deverá ser preenchido com o número do processo de credenciamento, sem o dígito verificador. Para serviço de "Vistoria de veículos de escolta" este campo deverá ser preenchido com o ano (dois algarismos), seguido do número da credencial (seis algarismos), seguido do RENAVAM do veículo (onze algarismos), mais um algarismo sequencial de um a nove referente a quantidade de vistorias realizadas no ano. Para serviço de "Teste de conhecimento para motorista de escolta" este campo deverá ser preenchido com o ano (dois algarismos), seguido do CPF do candidato (onze algarismos), mais um algarismo sequencial de um a nove referente a quantidade de testes realizados no ano. Para serviço de "Escolta com viatura PRF" este campo deverá ser preenchido com a data/hora da solicitação, iniciando com o dia (dois algarismos), seguido do mês (dois algarismos), seguido do ano (dois algarismos), seguido da hora (dois algarismos), seguido do minuto (dois algarismos), seguido do número da AET (sem a barra "/" e sem a letra "E").
